# 三体文字游戏 / Three-Body Text Game

基于《三体》小说的互动文字冒险游戏。

## 版本

- `三体.py` - 命令行版本
- `三体_gui.py` - GUI 版本 v1
- `三体_gui_v2.py` - GUI 版本 v2（最新）

## 运行

```bash
python 三体_gui_v2.py
```

## 游戏文本

- `游戏文本_1.txt` - 第一章
- `游戏文本_2.txt` - 第二章
- `游戏文本_3.txt` - 第三章
- `游戏文本_4.txt` - 第四章
- `决策树.txt` - 游戏决策树结构

## 依赖

- Python 3.x
- tkinter (GUI 版本)
