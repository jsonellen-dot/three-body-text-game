# 黑暗纪元四分支设计（修订版）

## 总体架构

```
威慑纪元 (deterrence_era_3)
  └─ 选择7（威慑失败）→ 黑暗纪元 (dark_era_1)
                              ↓
                    ┌─────────┼─────────┐
                    ↓         ↓         ↓         ↓
            1.黑暗结局  2.流浪纪元  3.解放纪元  4.地下纪元
           (直接结局)  (wanderer)  (liberation) (underground)
```

---

## 黑暗纪元（共同起点）

### dark_era_1: 威慑崩溃
**剧情：**
```
公元2270年，程心成为执剑人。

三体舰队突然加速。
智子发出最后通牒："交出引力波发射器。"

程心面对按钮，手指颤抖。
她想起了云天明，想起了那些三体孩子...

15分钟后，她放下了手。

"我不能毁灭两个世界..."

威慑失败。

水滴摧毁了所有引力波发射器。
三体舰队降落地球。

人类的黄金时代结束了。
黑暗纪元开始。
```

**选择：**
1. 【撤往澳洲】组织残余力量撤往澳大利亚保留区
   → dark_era_2_australia
   
2. 【转入地下】放弃地表，转入地下世界
   → dark_era_2_underground
   
3. 【绝望投降】接受三体统治，不再反抗
   → 结局：永久奴役（黑暗结局分支1）

---

### dark_era_2_australia: 澳洲保留区
**剧情：**
```
三体人将30亿人类驱赶到澳大利亚。

这是人类最后的保留区：
- 面积：770万平方公里
- 人口：30亿（全球幸存者）
- 资源：严重短缺，每天饿死数万人
- 监控：智子无处不在

三体人占领了其他所有大陆。
地球资源被疯狂开采，运往三体世界。

在悉尼废墟中，抵抗组织秘密集结。
他们发现了一个秘密：
三体人的工厂防御并不严密...
```

**选择：**
1. 【窃取技术】潜入三体工厂，窃取飞船技术
   → dark_era_3_tech_theft（流浪纪元前置）
   
2. 【寻找武器】寻找能威慑三体的终极武器
   → dark_era_3_weapon（解放纪元前置）
   
3. 【社交博弈】与三体温和派接触，寻求合作
   → dark_era_3_diplomacy（解放纪元前置）
   
4. 【放弃抵抗】接受现实，苟且偷生
   → 结局：人类保留区（黑暗结局分支2）

---

### dark_era_2_underground: 转入地下
**剧情：**
```
地表已经不安全。

在全球各地，人类开始向地下转移：
- 中国：利用废弃的矿井和地铁
- 美国：改造核掩体和地下城
- 欧洲：在阿尔卑斯山下挖掘洞穴
- 火星：殖民地转入地下

三体人控制地表，人类退守地下。

在地下世界，人类建立了新的文明：
- 地下农场（人工光源）
- 地下工厂（秘密生产武器）
- 地下城市（数亿人口）

但资源有限，空间狭小...
人类能在地下生存多久？
```

**选择：**
1. 【长期坚守】在地下建立永久文明
   → dark_era_3_underground_city（地下纪元前置）
   
2. 【地下反击】从地下发动突袭
   → dark_era_3_underground_war（解放纪元前置）
   
3. 【资源耗尽】地下资源耗尽，文明崩溃
   → 结局：地下灭绝（黑暗结局分支3）

```
孩子们从出生就生活在铁丝网内。
他们不知道地球曾经属于人类。

一千年后，人类成为了三体文明的附庸种族。
地球历史被改写，人类文明被遗忘。

这就是黑暗的永恒。
```

## 分支 2：流浪纪元

### dark_era_3_tech_theft: 技术窃取
**剧情：**
```
抵抗组织制定了"普罗米修斯计划"：
潜入三体工厂，窃取飞船技术。

经过三年的准备：
- 收买了三体工厂的人类劳工
- 绘制了工厂的完整地图
- 训练了突击队

行动开始！

突击队潜入工厂，复制了技术数据：
✓ 曲率驱动引擎
✓ 反物质能源
✓ 生命维持系统

但警报响起...
```

**选择：**
1. 【立即撤退】带着数据逃离
   → wanderer_era_1
   
2. 【继续窃取】冒险获取更多技术
   → 50%成功 → wanderer_era_1 / 50%失败 → 结局：行动失败

---

### wanderer_era_1: 秘密建造
**剧情：**
```
【流浪纪元开始】

在澳洲内陆的秘密基地，
人类工程师夜以继日地工作。

窃取的技术被破解、改进、应用。

五年后：
- 建成战舰：50艘
- 载人能力：每艘2万人，共100万人
- 航程：可达半人马座α星以外

但这只是全人类的0.03%...
剩下的29.9亿人怎么办？

舰队指挥官面临艰难抉择...
```

**选择：**
1. 【抽签登船】公平抽签，100万幸运儿
   → wanderer_era_2_lottery
   
2. 【精英优先】带走科学家、工程师、军人
   → wanderer_era_2_elite
   
3. 【全民建造】继续建造更多飞船，拖延时间
   → wanderer_era_2_delay（可能被发现）

---

### wanderer_era_2_lottery: 大逃亡
**剧情：**
```
抽签结果公布。

100万人登船，29.9亿人留下。

发射场上，哭声震天：
"带我走！求你们！"
"我的孩子中签了，但我没有..."
"为什么是他们，不是我？"

三体人发现了舰队！
水滴正在赶来！

舰队指挥官下令："立即起飞！"

引擎点火，战舰升空。
身后，地球越来越小。

那些留下的人，在地面上仰望天空...
```

**选择：**
1. 【寻找新家园】前往适居星球
   → 结局：新人类文明
   
2. 【游击战】在星际间打游击
   → 结局：星际流浪者
   
3. 【回归复仇】积蓄力量，百年后反攻
   → 结局：复仇之战

---

### wanderer_era_2_elite: 精英主义
**剧情：**
```
你选择了精英优先。

登船名单：
- 科学家：30万
- 工程师：20万
- 军人：20万
- 政府官员：10万
- 艺术家、医生等：20万

普通人被排除在外。

当真相泄露，澳洲陷入暴动：
"凭什么只有精英能走？"
"我们也是人类！"

军队镇压了暴动。
舰队在血泊中起飞。

船上的人永远无法忘记：
他们是踩着同胞的尸体逃走的...
```

**选择：**
1. 【负罪前行】带着愧疚建立新文明
   → 结局：精英的原罪
   
2. 【自我救赎】发誓要让新文明更公平
   → 结局：理想新世界

---

### wanderer_era_2_delay: 被发现
**剧情：**
```
你选择继续建造飞船。

"再等一年，我们能多救500万人！"

但三体人发现了基地。

水滴从天而降。

基地被摧毁，飞船被击毁。
100万人的希望化为灰烬。

抵抗组织全军覆没。

三体人冷笑："愚蠢的人类，
你们以为能瞒过智子吗？"
```

**结局：** 流浪计划失败

## 分支 3：解放纪元

### dark_era_3_weapon: 寻找终极武器
**剧情：**
```
科学家提出了一个疯狂的想法：

"我们可以制造'宇宙广播器'！
向全宇宙广播三体世界的坐标！

这样，三体人就会被其他文明摧毁。
我们重新获得威慑能力！"

但这需要：
- 大量能源（需要窃取三体的反物质）
- 精密设备（需要潜入三体实验室）
- 时间（至少3年）

风险极高，但这是唯一的希望...
```

**选择：**
1. 【执行计划】冒险制造广播器
   → liberation_era_1
   
2. 【放弃计划】风险太大，寻找其他办法
   → dark_era_3_diplomacy

---

### dark_era_3_diplomacy: 社交博弈
**剧情：**
```
情报显示：三体文明内部并非铁板一块。

有一派三体人（"和平派"）认为：
"殖民地球是错误的。
我们应该与人类共存，而非奴役。"

通过秘密渠道，你联系上了和平派领袖。

他说："我可以帮你们，但需要筹码。
如果你们能证明人类值得尊重，
我会说服三体议会给予你们自治权。"

这是一场危险的博弈...
```

**选择：**
1. 【展示价值】向三体展示人类文化和科技
   → liberation_era_1（外交路线）
   
2. 【不信任】这可能是陷阱，拒绝合作
   → 回到 dark_era_2_australia

---

### liberation_era_1: 威慑重建
**剧情：**
```
【解放纪元开始】

奇迹发生了！

【武器路线】
宇宙广播器建成！
你向三体发出最后通牒：
"立即撤离地球，否则我们广播你们的坐标！"

【外交路线】
三体和平派政变成功！
新政府宣布："我们将归还地球主权。"

无论哪条路线，结果是：
- 三体舰队撤离
- 人类重获地球控制权
- 亚洲、欧洲、美洲、非洲解放
- 澳洲保留区解散

人类欢呼雀跃。
黑暗纪元结束，解放纪元开始！

但新的危机正在酝酿...
```

**选择：**
1. 【重建地球】专注于恢复地球文明
   → liberation_era_2_rebuild
   
2. 【防范三体】保持警惕，防止三体反扑
   → liberation_era_2_defense
   
3. 【宇宙探索】利用窃取的技术探索宇宙
   → liberation_era_2_explore

---

### liberation_era_2_rebuild: 重建家园
**剧情：**
```
地球满目疮痍。

三体人留下的是：
- 被开采殆尽的矿山
- 被污染的河流
- 被摧毁的城市
- 30亿人口的创伤

重建需要一个世纪。

但人类充满希望。
孩子们在废墟中玩耍，
工人们在重建城市，
农民们在恢复农田...

然而，宇宙深处，
一个更强大的文明注意到了地球的广播...
```

**选择：**
1. 【继续重建】无视威胁，专注重建
   → 结局：短暂的和平
   
2. 【准备迎战】建造防御系统
   → liberation_era_3_war

---

### liberation_era_3_war: 新的威胁
**剧情：**
```
警报响起！

一个未知文明的舰队正在接近太阳系！

他们发来信息：
"我们收到了你们的广播。
你们暴露了自己的位置。
根据黑暗森林法则，你们必须被清除。"

人类面临比三体更强大的敌人...
```

**结局：**
- 选择1：【死战到底】→ 结局：人类的最后一战
- 选择2：【再次逃亡】→ 结局：永恒的流浪者
- 选择3：【谈判求和】→ 结局：新的奴役

## 分支 4：地下纪元

### dark_era_3_underground_city: 地下文明
**剧情：**
```
【地下纪元开始】

人类放弃了地表。

在地下，新的文明诞生：

**地球地下城市：**
- 北京地下城：5000万人口
- 纽约地下城：3000万人口
- 伦敦地下城：2000万人口
- 上海地下城：4000万人口
- 总计：全球50个地下城，20亿人口

**火星地下殖民地：**
- 奥林帕斯地下城：500万人口
- 水手谷地下城：300万人口

地下世界自给自足：
- 地下农场（LED人工光源）
- 地下工厂（秘密生产武器）
- 地下学校（培养新一代）

三体人控制地表，人类统治地下。

这是一种奇特的共存...
```

**选择：**
1. 【长期坚守】在地下永久生存
   → underground_era_2_permanent
   
2. 【地下反击】积蓄力量，从地下反攻
   → underground_era_2_counterattack
   
3. 【地下扩张】向地球深处挖掘
   → underground_era_2_deep

---

### underground_era_2_permanent: 地下王国
**剧情：**
```
一百年过去了。

地下文明已经完全成熟：
- 人口增长到30亿
- 科技独立发展
- 文化独具特色

地下人类已经适应了没有阳光的生活。
新一代人从未见过地表。

他们不知道天空是什么颜色，
不知道风是什么感觉，
不知道雨是什么味道...

但他们很幸福。

地下就是他们的世界。
```

**结局：**
- 选择1：【永居地下】→ 结局：地下王国
- 选择2：【重返地表】→ underground_era_3_return

```
"我们不能永远躲在地下！"

抵抗军从地下发动突袭：
- 从地铁隧道突然出现
- 从矿井涌出
- 从地下工厂冲出

三体人措手不及！

人类夺回了部分地表城市：
✓ 北京
✓ 上海
✓ 纽约
✓ 伦敦

但三体舰队正在集结...
```

### underground_era_2_deep: 深入地心
**剧情：**
```
科学家提出了疯狂的计划：

"我们可以向地球深处挖掘！
在地幔层建立城市！

那里有：
```
